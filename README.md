# mangaSeeFeeder

<https://mangasee123.com/>のサブスクリプションリストから、読み終わっていないデータを
抽出してメール送信する。

# need for run

+ setup.pyの実行
  + 設定データの配置
+ 対応のchromedriver.exe
+ setLogin.pyの実行
+ main.pyの実行


# main functions

## main.py

スクレイピングからメール送信を一連して実行する。予定。

## setup.py

初期のフォルダ作成などを行う。
初回のみ実行。

## setLogin.py

seleniumのchromeProfileにログイン情報を記憶させるため。
初回のみ実行。
